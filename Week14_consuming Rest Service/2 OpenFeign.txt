------------------------------------------------------------------------------

@SpringBootApplication
@EnableFeignClients(basePackages = "com.eazybytes.eazyschool.proxy")
public class EazyschoolApplication {

	public static void main(String[] args) {
		SpringApplication.run(EazyschoolApplication.class, args);
	}

}

1.2 : Create ContactProxy interface to call the service API provider 

@FeignClient(name = "contact", url = "http://localhost:8080/api/contact",
    configuration = ProjectConfiguration.class)
public interface ContactProxy {

    @RequestMapping(method = RequestMethod.GET, value = "/getMessagesByStatus")
    @Headers(value = "Content-Type: application/json")
    public List<Contact> getMessagesByStatus(@RequestParam("status") String status);

}

Explain : 

- name = "contact" : Bean name for the FeignClient 
- url = "http://localhost:8080/api/contact" : UPL from service API provider 
- configuration = ProjectConfiguration.class : we we put our username and passport 
- interface ContactProxy : we create interface to handle our request 

+ @RequestMapping(method = RequestMethod.GET, value = "/getMessagesByStatus") : Request mapping to the 
API that they provide to use URL 
+ @Headers(value = "Content-Type: application/json") : There header they senting 
+  public List<Contact> getMessagesByStatus(@RequestParam("status") String status) : our method in the client to handle the request API 

The parameter need to be the same singture as the API provider service
