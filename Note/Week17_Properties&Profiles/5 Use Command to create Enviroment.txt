Gold: we want to activate the properties Dev, uat or prod from outside by using command 

1. Create Jar or Var file 

>> mvn clean install -Dmaven.test.skip=true

==> the file in the folder target 

2. Runing the jar filed 

go to the file location then run below command to run with our different properties : 

>> mvn spring-boot:run "-Dspring-boot.run.profiles=dev"

>> mvn spring-boot:run "-Dspring-boot.run.profiles=prod"

3. How to run .jar file 

>> java -jar yourfile.jar
