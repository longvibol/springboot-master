Globle Exception Handdler is what we want to show to user when it have error happen inside the spring 

1. Setup GlobalExceptionController

in the controller : we create class : GlobalExceptionController

import lombok.extern.slf4j.Slf4j;

@Slf4j
@ControllerAdvice
public class GlobalExceptionController {
	
	@ExceptionHandler(Exception.class)
	public ModelAndView exceptionHandler(Exception exception) {
		ModelAndView errorPage = new ModelAndView();
		errorPage.setViewName("error");
		errorPage.addObject("errormsg",exception.getMessage());
		return errorPage;
	}
	

}

Note: 

- First we need to add : @ControllerAdvice : to tell spring boot we redirect to our customer Expection handler 
- add :  @ExceptionHandler(Exception.class) : in out own expection we want to catch : 
- ModelAndView errorPage = new ModelAndView(); what we want to show to out FE
