We can group our properties with the profile 

- Dev
- uat
- prod 

+ Dev (Development) is where developers write and test new code. 

+ UAT (User Acceptance Testing) is a stage where end-users test the application to ensure it meets their needs.

+ Prod (Production) is the live environment where the application is available to the general public. 

-----------------------------------------------------
